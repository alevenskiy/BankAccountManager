# BankAccountManager

Task 1

Прототип банковской системы, которая содержит информацию о клиентах и их счетах.  
Счётом любого клиента можно управлять:  
	-	(создание клиента;)  
	-	(клиенты хранятся в json в корне проекта;)  
	-	открывать и закрывать счета;  
	-	осуществлять переводы с одного счёта на другой;  
	-	(переводы осуществляются со счета на счет внутри клиента или со счета клиента на первый счет другого клиента)  
Для этого необходимо использовать обобщение и параметризированные классы.
