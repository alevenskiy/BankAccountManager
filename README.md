# BankAccountManager

Task 1

Прототип банковской системы, которая содержит информацию о клиентах и их счетах\n
Счётом любого клиента можно управлять:\n
	-	открывать и закрывать счета;\n
	-	осуществлять переводы с одного счёта на другой;\n
Для этого необходимо использовать обобщение и параметризированные классы\n
\n
	+	создание клиента\n
	+	клинты хранятся в json в корне проекта\n
	+	переводы осуществляются со счета на счет внутри клиента или со счета клиента на первый счет другого клиента\nS
